# GPS Visualizationtool
A tool to visualize and trace gps data extracted from csv files

## How to run
1. To install the necessary packages run the command "npm install".

2. To run the program either use the command "npm start" or "node app.js".

3. Now that it's running locally open a webbrowser like Firefox, Google Chrome or Microsoft Edge and navigate to http://localhost:3000/ to view the map.

4. Make sure to upload a CSV file with the following columns:
  •	Latitude
  •	Longitude
  •	Name
  •	UTC Date
  •	UTC Time
  •	Speed
  •	Course
  •	Sats
  •	HDOP
  •	CellID
  •	LAC
  •	Name2
  •	TA
  •	PL
  •	M
  •	IntPwr
  •	ACC
  •	Temp
  •	Charging
  •	RSSI
  •	Mode
  •	CAP
